
# --------------------
# Type List  
# [Init]       : 초기 생성    
# [Feat]       : 기능 (새로운 기능)
# [Fix]        : 버그 (버그 수정)
# [Design]     : CSS 등 사용자 UI 디자인 변경
# [HOTFIX!]    : 급한 버그 수정(주로 출시버전, D-0 라벨표기)
# [Refactor]    : 코드 리팩토링
# [Comment]    : 필요한 주석 추가 및 변경    
# [Style]      : 스타일 (코드 형식, 세미콜론 추가: 비즈니스 로직X)  
# [Docs]       : 문서 추가, 수정, 삭제(주로 README)    
# [Test]       : 테스트 (테스트 코드 추가, 수정, 삭제: 비즈니스 로직X)    
# [Chore]       : 기타 (빌드 스크립트 수정, 패키지 매니저 변경, assets 변경 등)    
# [Rename]     : 파일 혹은 폴더명 수정 또는 이동 작업    
# [Add]      : 파일을 추가하는 작업만 수행한 경우
# [Remove]      : 파일을 삭제하는 작업만 수행한 경우 
# [Deprecated] : 더 이상 사용되지 않는 코드나 기능을 제거
# [Security]   : 보안 관련 코드 추가 및 수정
# [Merge]      : 팀원들과 동기화 / 충돌 코드 병합
# [Deploy]     : 배포 관련
# --------------------
